# handlebars-nodejs-template
A quickstart template file for a simple NodeJS Express App w/ Handlebars

## A few things you need to change...
Check the packange.json file and change the following stuff:

### The Project Info
Mostly just the project name, but feel free to change the other stuff.

```
{
    "name": "insert-project-name-here",
    "version": "1.0.0",
    "description": "",
    ...
}
```

### GitHub Links
Just figure out the format yourself. I know you can do it!

```
{
    ...
    "homepage": "https://github.com/notgian/insert-repo#readme", //Change me
    "bugs": {
        "url": "https://github.com/notgian/insert-repo/issues" // Change me
    },
    "repository": {
        "type": "git",
        "url": "git+https://github.com/notgian/insert-repo.git" // Change me
    },
    ...
}
```
